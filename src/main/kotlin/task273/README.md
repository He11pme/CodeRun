# 273. Курьер Василий

**Сложность**: Средняя

***Время выполнения: - 240 мс; Память: - 32,48 МБ***

## Описание

Юный программист Василий на каникулах решил подработать. В силу своего возраста он смог стать только курьером. Компания, в которой он теперь работает, находится на прямой улице бесконечной длины. Суть работы Василий — разносить еду пёсикам по всем домам, находящимся на этой улице.

Дома имеют целочисленные координаты, а главный офис находится в координате (S). Там Василий получает **n** пакетов с едой, на каждом есть адрес: координата дома, куда нужно доставить пакет. Василий очень ленивый мальчик, ему хочется пройти как можно меньше. Напишите программу, которая определит по заданному местоположению офиса и домов **минимальное расстояние**, которое необходимо пройти Василию, чтобы разнести всю еду.

## Формат ввода

В первой строке находятся числа:
- s, ( |s| <= 10^8 ) — координата, откуда Василий начинает разносить еду.
- n, ( 1 <= n <= 100 000 ) — количество пакетов.

Во второй строке находятся ai, ( |ai| <= 10^8 ) — координаты домов, куда нужно доставить еду.

## Формат вывода

Выведите единственное число, равное **минимальной длине пути**, который должен пройти Василий, чтобы разнести всю еду.

## Ограничения

**Ограничение времени**: 2 с  
**Ограничение памяти**: 256 МБ
