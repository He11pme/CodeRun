# Solving Yandex CodeRun Tasks - Kotlin (Mobile Development)
This repository contains my solutions to tasks from **Yandex Coderun**, specifically from the **"Mobile Development"** section.
All solution are written in **Kotlin**, with a focus readability and common Kotlin development pracices.
## About the Project
- Source: https://coderun.yandex.ru/catalog?language=kotlin&groups=mobile
- Goal: improve algorithmic thinking and strengthen Kotlin skills
- Each task is organized in its own package and typically includes:
  - the task statement
  - the Kotlin solution
  - comments explaining key parts of the implementation
 ## Technologies & Concepts
 - **Kotlin**
 - Use of basic and advanced languages features (when needed):
  - higher-order functions
  - working with collections and strings
  - algorithms and common data structures
  - extension functions
  - data classes, sealed classes
